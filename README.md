## Project Overview

In this project, I explored the fundamentals of Transformer-based language models by following 
Andrej Karpathy's **"Let's Build GPT: From Scratch"** video.

I learned how a Transformer processes text, starting from raw Shakespeare data and converting it 
into training examples where the model is given an input sequence and learns to predict the next 
token. I gained an understanding of how token embeddings, self-attention, feed-forward networks, 
and the training process work together to enable next-token prediction.

This project focuses on building the core concepts behind GPT from scratch rather than using high-level 
libraries. It provides a practical understanding of how modern large language models are trained and how 
they generate coherent text by learning patterns from data.
